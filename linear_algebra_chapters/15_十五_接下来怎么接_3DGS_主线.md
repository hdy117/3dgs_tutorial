#### 2. **知识关联题**
把下面的 3DGS 概念和对应的线性代数工具连线：

| 3DGS 概念 | 线性代数工具 |
|----------|------------|
| 高斯椭球的方向和尺度 | A. Jacobian 矩阵 |
| 透视投影的局部近似 | B. 协方差矩阵的特征分解 |
| 从世界坐标到相机坐标 | C. 旋转矩阵 + 齐次坐标 |
| 屏幕 footprint 的计算 | D. 投影矩阵 + 缩放 |

<details>
<summary>👉 点击查看答案</summary>

**答案：**
- **高斯椭球的方向和尺度 → B**. $\Sigma = R \text{diag}(s^2) R^T$通过特征分解得到主轴方向和标准差
- **透视投影的局部近似 → A**. $u \approx u_0 + J_{11}\delta x + J_{12}\delta y + J_{13}\delta z$
- **从世界坐标到相机坐标 → C**. $x_{\text{cam}} = R x_{\text{world}} + t$,用齐次坐标写成矩阵乘法
- **屏幕 footprint 的计算 → D**. $\Sigma_{2d} \approx J \Sigma_{\text{cam}} J^T$

**关键公式记忆：**
```python
# 世界→相机
x_cam = R @ x_world + t

# 局部→世界（高斯旋转）
R_gaussian = quaternion_to_matrix(quat)

# 投影 Jacobian
J = d(proj)/d(x)

# 屏幕协方差
Sigma_2d = J @ Sigma_cam @ J.T
```

</details>

#### 3. **自测题：核心问题清单**
下面这些问题，如果你能独立回答出"为什么"和"怎么做",说明你已经准备好了:

<details>
<summary>👉 点击查看答案（答案要点）</summary>

**Q1: 为什么协方差矩阵可以编码椭球？**  
→ $\Sigma = R \text{diag}(s^2) R^T$,特征分解给出主轴方向 $R$和尺度$s$

**Q2: 为什么要用齐次坐标？**  
→ 把平移也写成矩阵乘法，实现"旋转 + 平移"的统一链：$[R|\mathbf{t}; \mathbf{0}^T|1]$

**Q3: Jacobian 在 3DGS 里什么时候用到？**  
→ 透视投影局部线性化：$\delta p \approx J \delta x$,协方差传播 $\Sigma_{2d} = J \Sigma_{\text{cam}} J^T$

**Q4: 行列式为 0 时高斯会发生什么？**  
→ $\det(\Sigma)=0 \Rightarrow$ 椭球压扁成平面或线，失去三维意义，渲染可能异常

**Q5: 为什么特征分解要求对称矩阵？**  
→ 只有实对称矩阵才能保证：①所有特征值是实数；②特征向量正交；③可以写成 $P\Lambda P^T$

**Q6: SVD 和特征分解的区别？**  
→ SVD 适用于任意矩阵（包括非方、不对称）,特征分解只要求可对角化方阵。SVD = UΣVᵀ总是存在！

**如果以上问题你都能自己推导，那你已经真正掌握了线性代数在 3DGS 中的核心用法！**

</details>
