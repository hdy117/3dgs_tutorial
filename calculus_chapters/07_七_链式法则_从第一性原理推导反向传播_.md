# Ch07 - 链式法则：从第一性原理推导反向传播 🔥

> **本章目标**：理解为什么 `loss.backward()` 能穿透渲染管线，把梯度传回每个 Gaussian 参数。  
> **核心洞察**：**可微分渲染 = 嵌套函数的导数可以用链式法则求**。这是 3DGS（以及所有深度学习方法）的基石。

---

## 🎯 起点：为什么需要链式法则？

### 问题场景

render() 是三层嵌套函数：
```text
Gaussian 参数 θ → 投影 f₁(θ) → 排序 f₂(f₁) → Alpha blending f₃(f₂) → Loss L(f₃)
```

**关键矛盾**：你想求 `dL/dθ`，但 L 和 θ 之间隔了三层函数！怎么办？

### 链式法则的答案（一维版）

$$\frac{d}{dx} f(g(h(x))) = f'(g·h) · g'(h) · h'(x)$$

**工程翻译**：从 output 往 input 传，每一层乘以这层的导数。

---

## 🔥 Part 1: 从第一性原理推导链式法则（一维）

### 证明思路（不需要死记硬背，但要懂逻辑）

设 `y = f(u)`, `u = g(x)`。想求 `dy/dx`。

**Step 1**: 用导数定义：
$$\frac{dy}{dx} = \lim_{Δx→0} \frac{f(g(x+Δx)) - f(g(x))}{Δx}$$

**Step 2**: 分子分母同时除以 `Δu = g(x+Δx) - g(x)`：
$$\frac{dy}{dx} = \lim_{Δx→0} \underbrace{\frac{f(u+Δu) - f(u)}{Δu}}_{→ f'(u) \text{ 当 } Δu→0} · \underbrace{\frac{g(x+Δx) - g(x)}{Δx}}_{→ g'(x)}$$

**Step 3**: 当 `Δx→0`，也有 `Δu→0`（假设 g 连续），所以：
$$\boxed{\frac{dy}{dx} = \frac{df}{du} · \frac{dg}{dx}}$$ ✅

### ✅ 关键洞察

> **链式法则的直觉**：总变化率 = (外层对中间变量的变化率) × (中间变量对内层的变化率)。  
> 就像"车速 = (油门开度 → 加速度的关系) × (时间 → 油门的关系)"。

---

## 🔥 Part 2: 扩展到多维（Jacobian 矩阵）

### 从标量到向量

在 3DGS 中：
- θ 是向量（比如 `[μ_x, μ_y, α]`，维度 n=3）
- 投影后的参数 u = f₁(θ) 也是向量（维度 m=5，可能包含 2D 位置 + size）

**问题**: `∂L/∂θ` 怎么算？（L 是标量，θ 是向量 → 梯度向量）

### Jacobian 矩阵的定义

```text
J = ∂u/∂θ = [ ∂u₁/∂θ₁   ∂u₂/∂θ₁   ...   ∂u_m/∂θ₁
              ∂u₁/∂θ₂   ∂u₂/∂θ₂   ...   ∂u_m/∂θ₂
              ⋮          ⋮         ⋱    ⋮
              ∂u₁/∂θₙ   ∂u₂/∂θₙ   ...   ∂u_m/∂θₙ ]  (m×n 矩阵)
```

**链式法则（向量形式）**:
$$\frac{∂L}{∂θ} = J^T · \frac{∂L}{∂u}$$

（注意转置！因为 `∂L/∂θ` 是 n×1，`J` 是 m×n，`∂L/∂u` 是 m×1）

## 🔥 Part 3: 3DGS 中的完整反向传播路径

### 渲染管线的链式法则展开

```text
Loss = MSE(rendered_img, ground_truth)
         ↓ ∂L/∂pixel (简单：就是误差本身，对每个像素是标量)
         
pixel_color[r,g,b] = alpha_blending(gaussians_2d_in_this_pixel)
         ↓ ∂color/∂(α_i, c_i) for each Gaussian i 贡献到这个 pixel
         （alpha blending 的导数：见下节）
         
gaussian_2D = project_3D_to_2D(gaussian_3D)
         ↓ ∂(α_2d, c_2d)/∂(μ_3D, Σ_3D)  ← Jacobian! 投影公式的导数
         
最终：∂L/∂θ_i = Σ_{pixels} (∂L/∂pixel · ∂pixel/∂g_i,2D · ∂g_i,2D/∂θ_i)
```

### ✅ 关键结论

> **可微分渲染的核心**：渲染管线的每一层都是可微的 → 整个 pipeline 可微！  
> - 投影公式（仿射变换）→ 线性函数，导数是常数矩阵
> - Alpha blending → 多项式组合，解析求导
> - Loss (L1/MSE/SSIM) → 标准 loss 函数，梯度有现成公式

## 🔥 Part 4: 实战例子 - Alpha Blending 的导数

### Alpha Blending 公式回顾

对单个像素，假设贡献的高斯按深度排序为 `g₁, g₂, ..., g_k`：
$$C = c_1·α_1 + c_2·(1-α_1)·α_2 + c_3·(1-α_1)(1-α_2)·α_3 + ...$$

**问题**: 求 `∂C/∂α_i`（对第 i 个高斯的透明度）？

### 手算导数（以 k=3 为例）

展开：
$$C = c_1 α_1 + c_2 α_2 - c_2 α_1 α_2 + c_3 α_3 - c_3 α_1 α_3 - c_3 α_2 α_3 + c_3 α_1 α_2 α_3$$

对 `α₂` 求导：
$$\frac{∂C}{∂α_2} = c_2(1-α_1) - c_3 α_3(1-α_1) = (c_2 - c_3 α_3)(1-α_1)$$

**工程意义**: 如果 `g₂` 的 alpha 增大，最终颜色会往哪个方向变？
- `(1-α₁)` → 被前面高斯遮挡的程度（α₁越大，影响越小）
- `(c₂ - c₃α₃)` → g₂自己的颜色和后面高斯的"净贡献差"
